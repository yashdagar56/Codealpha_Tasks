## 🌟 The Web Version
Experience modern AI aesthetics and interact with the bot instantly. No installation required.
Click the [Live Demo Link](https://yashdagar56.github.io/Codealpha_Tasks/codealpha_chatbot/index.html) to start chatting!

## 💻 The Python Version
The raw backend logic is written in pure Python.
1. Make sure Python is installed.
2. Open your terminal in this folder and run:
   ```bash
   python chatbot.py
   ```
3. Type `hello`, `how are you`, or `tell me a joke` to interact! Type `bye` to exit.
